This repository contains beginner-friendly Machine Learning projects built using Python and Scikit-learn. The projects demonstrate the fundamentals of Linear Regression, data handling, and visualization.
# House-Price-Prediction-
A simple Linear Regression model that predicts house prices based on:  
Area (sq ft)  
Number of bedrooms

  🔧 Tech Used:  
    Python 
    Pandas  
    Scikit-learn  
📊 Output:  
  Predicts house price based on input features  
Example: 
  Input: area = 2000, bedrooms = 3
  Output: ~55 lakh
